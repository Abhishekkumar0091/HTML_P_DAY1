11.What are Attributes in html

Answer

In HTML, attributes are special words used inside the opening tag of an element to provide additional information about that element.

1.id – Gives a unique identifier to an element.
2.class – Groups elements together for styling with CSS or scripting with JavaScript
3.src – Specifies the source of an image, video, or other media.
4.alt – Alternative text for images (shown if the image can’t load).

12.What is dns

Answer

DNS stands for Domain Name System.
It is like the phonebook of the internet.

How DNS Works (Step by Step):

You type www.example.com in your browser.
Your computer asks the DNS server: "What is the IP address of this domain?"
The DNS server looks it up and replies with an IP address (e.g., 93.184.216.34).
Your browser then connects to that IP address and loads the website.

13. what is ip address

Answer

IP (Internet Protocol) address is a unique number assigned to every device (computer, phone, server, etc.) on a network. It works like a home address for devices so they can send and receive data.

Types of IP addresses:
1.IPv4 – 32-bit, looks like:
192.168.1.1
2.IPv6 – 128-bit, created because IPv4 addresses are running out:
2400:cb00:2048:1::c629:d7a2

14.What is Localhost?

Answer

Localhost means “this computer” or “loopback address”.
It always points to the IP address 127.0.0.1 (for IPv4) or ::1 (for IPv6).

18.Difference between div and span

Answer

Difference between div and span
Feature div span
Type of Element Block-level element Inline element
Layout Behavior Starts on a new line and takes full width by default Stays in the same line, only takes as much width as needed
Use Case Used to group larger chunks of content (sections, containers, layouts) Used to style or highlight small parts of text or inline content
Styling Suitable for layout and structure with CSS Suitable for styling individual words, phrases, or inline content
Example Width/Height Can have width, height, margins, paddings applied easily Can have styles but width/height behave differently (inline)

20.What are self closing tags

Answer

A self-closing tag (also called void tag) is an HTML tag that does not need a closing tag because it doesn’t wrap any content.
For Example:- br,hr,img,input,meta,link etc.
