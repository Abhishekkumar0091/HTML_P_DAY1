4.What is dns and when it come in picture

Answer

What is DNS?

DNS stands for Domain Name System.
It works like the phonebook of the internet.
Humans remember names like www.google.com, but computers work with IP addresses like 142.250.182.206.
DNS converts (resolves) domain names into IP addresses so your browser can connect to the right server.

When does DNS come in picture?

DNS comes into play when you type a website URL in your browser.
Example: You type www.google.com in Chrome.
Your computer asks the DNS server:
"What is the IP address of www.google.com ?" The DNS server replies with something like 142.250.182.206.
Your browser then uses this IP to connect to Google’s server and load the page.

10.what does viewport meta tag do

Answer

The viewport meta tag controls how a webpage is displayed on mobile devices. By default, web pages were designed for desktops, so when you open them on a phone, they might look zoomed out or tiny. The viewport meta tag tells the browser how to scale and size the page for the screen.
meta name="viewport" content="width=device-width, initial-scale=1.0"
